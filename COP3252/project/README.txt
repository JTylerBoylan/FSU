Dungeon Crawler Game
Jonathan T. Boylan (SOLO)
COP3252

This game includes 3 preloaded levels with increasing difficulty.

The controls are:
 - Left: 'A'
 - Right: 'D'
 - Jump: 'W'
 - Attack/Shoot: '[SPACE]'

The goal is to reach the "treasure" at the end of each level, indicated by a gold colored square.
Enemies are colored red and will reset the level if touched.
You win when you have passed all 3 levels

NOTE: Walls are glitchy
